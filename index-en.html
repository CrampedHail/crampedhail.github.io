Harmonic Optimization Algorithm

GitHub

The project aims to implement a harmonic optimization algorithm for the knapsack problem invented by me and my colleagues. The very idea of the problem is inspired by computer games. The problem is to find the cheapest set of items, and the project's task is to find a set whose stats will be greater than those of your opponent. The screenshot shows the 10 item sets and their prices. The starting memory ("Pamięć początkowa") is 10 random sets and the final memory("Pamięć końcowa") is the top 10. As you can see in the screenshot above, the prices of the sets have been almost cut in half.
