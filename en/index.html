Harmonic Optimization Algorithm

GitHub

The project aims to implement a harmonic optimization algorithm for the knapsack problem invented by me and my colleagues. The very idea of the problem is inspired by computer games. The problem is to find the cheapest set of items, and the project's task is to find a set whose stats will be greater than those of your opponent. The screenshot shows the 10 item sets and their prices. The starting memory ("Pamięć początkowa") is 10 random sets and the final memory("Pamięć końcowa") is the top 10. As you can see in the screenshot above, the prices of the sets have been almost cut in half.

Java Mini-Games

GitHub

All mini-games were created in my spare time from studying in Java. Due to the simplicity of use, the Swing library was used to implement the GUI.

  • Battleships - a digital version of a turn-based game of battleships. Gameplay is available for only one person while your enemy is a simple bot. Player's and Enemy statistics are shown on the right side of a game window. There are 2 available interface languages - English and Polish. If needed you can turn on the gameplay help systems. Enemy's ships board can be shown, shots hints can be turned on and there is also a short tutorial on how to play.
  • Hangman Game - a turn-based game in which one of the players thinks of a word and the rest of the players tries to guess this word letter by letter or whole word at once. One bad letter means another hangman piece and the wrong word guess means the end of this round. Word bank can be edited so that it is never too easy.
  • Connect4 (Połącz4 in Polish) - digital version of the well-known board game "Line up 4" made by me and my college friend Kuba Swisłocki. Gameplay is fairly easy - the player decides which column wants to insert their colored 'token'. This token lands on the lowest free space in this column. The player can win by lining up 4 tokens - horizontally, vertically, or diagonally. Background color and token colors can be changed. Credits made for fun can look familiar to fans of a particular sci-fi movie saga
  • Tanks Game - a real-time game in which you shoot the enemy's tank. Each shot has a short cooldown. Every Tank has 3 health points and each hit takes 1 HP. After 3 hits enemy's tank respawns and you get one point.
  • Game of TicTacToe - it's definitely the simpliest game of my creation. Each of the players can be a computer enemy and if you set both to computer enemies they can play with each other.
